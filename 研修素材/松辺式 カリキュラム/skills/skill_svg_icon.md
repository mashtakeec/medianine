スライド内のSVGアイコンに関するルール。

**サイズ体系:**
| 使用箇所 | サイズ |
|---|---|
| メインカードアイコン | 36px（大） |
| 中アイコン | 24px（中） |
| フッターアイコン | 14px（小） |
| ジャンルラベル内 | 16px（白色） |
| インライン（タイトル内） | 18〜20px |

**色ルール:**
- 通常背景（白・グレー）上: `fill="#C8232C"` / `stroke="#C8232C"`（赤）
- 赤背景・ダーク背景上: `fill="white"` / `stroke="white"`

**閉じタグ:**
- SVGの閉じタグは `/>` で必ず閉じる
- よくあるバグ: `opacity="0.15}/>` → `opacity="0.15"/>` に修正

**Why:** 閉じタグのミスはSVG全体が崩れる原因になる。色ルール違反はコントラスト不足で文字が読めなくなる。
