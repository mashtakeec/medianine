---
name: 改行制御スキル
description: スライド内テキストの改行位置を<br>タグで明示的に制御する手法
type: feedback
---

スライド内のテキストはCSSの自動折り返しで意図しない位置で改行されるため、`<br>`タグで意味の切れ目に明示的に改行を入れる。

**Why:** CSSの自動折り返しは文字数ベースで改行するため、「助詞の直後」「中黒の前」など読みにくい位置で切れてしまう。

**How to apply:**
1. ユーザーが改行位置を指定 → そのとおりに`<br>`を挿入
2. 修正対象のスライド番号（Pxx）を特定し、srcdocを抽出→テキスト置換→再エンコードして書き戻す
3. Python の `html.unescape()` → 編集 → `html.escape(quote=True)` の手順を必ず守る

**パターン例:**
- `・`の後で改行: `検索意図・目次構成・<br>ツール設定`
- 句点の後で改行: `AIは万能ではない。<br>使いこなす人のスキルが結果を決める`
- 文の意味の区切りで改行: `YouTube台本市場を理解し、<br>3ジャンルの構成パターンを<br>習得する`

**既存の`<br>`位置変更もある:**
- `初心者が狙える3ジャンル・相場<br>・収益ロードマップ` → `初心者が狙える3ジャンル・相場・<br>収益ロードマップ`（中黒を前の行に残す）
