# 動画生成サイト 設計報告書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | 動画生成サイト |
| 目的 | AIエージェントを活用した動画の自動生成・編集 |
| コア技術 | Remotion（プログラマブル動画生成フレームワーク） |
| 出力形式 | MP4（H.264、1920×1080、60fps） |

## 4. コンポーネント設計

### 4.1 コンポジション構成（MyComposition）

動画は以下のレイヤーで構成される（下から上の重ね順）：

```
┌──────────────────────────────────────────┐
│  ④ 字幕（Subtitles）         ← 画面下部  │
│  ③ 情報ラベル                ← 左上/左下  │
│  ② ヘッダー（Header）        ← 右上      │
│  ① ロゴ（Logo）              ← 左上      │
│  ── グラデーションオーバーレイ ──          │
│  ⓪ 背景動画（background.mp4）            │
└──────────────────────────────────────────┘
```

### 4.2 コンポーネント一覧

| コンポーネント | ファイル | 役割 |
|----------------|----------|------|
| `RemotionRoot` | Root.tsx | ルート定義。コンポジションID・解像度・fps・尺を設定 |
| `MyComposition` | Composition.tsx | メインコンポジション。全レイヤーの合成 |
| `Logo` | Logo.tsx | 左上に表示するロゴ画像（白背景＋金枠） |
| `Header` | Header.tsx | 右上のタイトル＋サブタイトル（ニュース風UI） |
| `Subtitles` | Subtitles.tsx | フレームベースの字幕表示。時間に応じて切り替え |

### 4.3 字幕データ構造

```typescript
type Subtitle = {
  text: string;       // 表示テキスト
  startFrame: number; // 表示開始フレーム
  endFrame: number;   // 表示終了フレーム
};
```

現在の字幕データ（4件）：

| # | テキスト | 開始フレーム | 終了フレーム | 時間（秒） |
|---|----------|-------------|-------------|-----------|
| 1 | 動画編集が驚くほど簡単になりました | 0 | 240 | 0〜4s |
| 2 | AIエージェントを使えば、数分で完成です | 240 | 480 | 4〜8s |
| 3 | ロゴや見出しも、あなたの思い通りに | 480 | 720 | 8〜12s |
| 4 | 次世代のクリエイティブ体験を今すぐ | 720 | 2270 | 12〜37.8s |

## 8. デザイン方針

- **ニュース番組風UI**: ヘッダーに白背景タイトル＋黄色（#f0c300）のサブタイトルバー
- **視認性重視**: グラデーションオーバーレイで背景動画上のテキストを読みやすく
- **字幕スタイル**: 太字・黒ストローク付き白文字（テレビ風テロップ）
