テキストが枠内で2行に折り返されている場合、1行に収める方法。

**Why:** カード内の説明文が2行になると帯のバランスが崩れ、他の項目との高さが不揃いになる。

**How to apply:**

**方法1: テキスト自体を短縮する**
- `「このクライアントの型」` → `「クライアントの型」`（「この」を削除）
- `P25で作成したNotebookLMの` → `作成したNotebookLMの`（不要な参照を削除）
- `Gem＝自分専用AIの設定ファイル` → `Gem＝自分専用の設定ファイル`

**方法2: フォントサイズを下げる**
- `font-size:13px` → `font-size:12px`（card-descなど）
- P34/P35で実施：5項目の帯バランスを揃えるために統一

**方法3: CSSで折り返しを制御する**
```css
.cmp-text { white-space: nowrap; }
```

**判断基準:**
- 意味が変わらない範囲でテキスト短縮が最優先
- 短縮できない場合はフォントサイズ調整
- `white-space:nowrap`は確実に1行だが、枠からはみ出すリスクあり
